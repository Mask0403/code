`软链接`和`硬链接`的区别
本质和特点不同
1删除软硬链接,源文件都还存在
2删除源文件,只有硬链接还存在
3只有软链接可以指向目录和跨分区
ps.真实场景大多数使用软链接,软链接无法二次覆盖创建

```
1使用``或者${}同步执行
```

cp和mv有一个特殊的选项`-t`,用来调换源参数和目标参数
```
old> cp /etc/chrony.conf(源文件) /opt/chrony.conf(目标文件)
new> cp -t /opt/chrony.conf(目标文件) /etc/chrony.conf(源文件)
```
